# Paper Review: ParallelKWS (IEEE SPL 2024)

**Bridging Audio and Text using Parallel-attention for UDKWS**

| Field | Value |
|---|---|
| **Authors** | Youkyum Kim, Jaemin Jung, Jihwan Park, Byeong-Yeol Kim, Joon Son Chung |
| **arXiv** | 2408.03593 |
| **Method** | Parallel self-/cross-attention + phoneme duration alignment loss |
| **Enrollment** | Text-only (zero-shot) |
| **Results** | SOTA on multiple benchmarks, no extra training data |
| **Relevance** | Medium — text-enrolled paradigm, different from audio enrollment |

**Key Takeaway:** Important for Related Work — text-enrolled UDKWS baseline. Phoneme duration alignment loss is a novel contribution for audio-text synchronization.
