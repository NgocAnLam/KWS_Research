# Paper Review: Synth4Kws (INTERSPEECH 2024 Workshop)

**Synthesized Speech for User Defined KWS in Low Resource Environments**

| Field | Value |
|---|---|
| **Authors** | Pai Zhu, Dhruuv Agarwal, Jacob W. Bartel, Kurt Partridge, Hyun Jin Park, Quan Wang |
| **arXiv** | 2407.16840 |
| **Method** | TTS-generated synthetic training data for custom KWS |
| **Results** | +30.1% EER, +46.7% AUC improvement with optimal TTS data |
| **Relevance** | High — synthetic data approach could help Vietnamese dataset collection |

**Key Takeaway:** Synthetic TTS data significantly improves KWS in low-resource settings. Important for Vietnamese where labeled data is scarce. Could use Vietnamese TTS (e.g., VietTTS) to augment our dataset.
